## Prerequisites

- PHP 8.1
- PostgreSQL 10.5

## Installation

1. Clone the GitHub repository:

   ```bash
   git clone https://github.com/Adpra/sekawan-media.git
   
2. Install the project dependencies using Composer:
    
   composer install

3. Run the database migrations:

  php artisan migrate

4. Seed the database with initial data:

  php artisan db:seed
